## user_guide

次目录为学习 `reportlab` 包时，摘录出来的生成官方用户手册的汉化版本。（机翻+人工翻）

需要环境：`python 3.6+` , 安装 `reportlab` 包

生成 `reportlab` 的用户手册请按下列方法执行：

```shell script
cd user_guid
python genuserguide.py
```
